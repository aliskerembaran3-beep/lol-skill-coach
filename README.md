# 🎮 LoL Skill Coach v2.0

League of Legends için AI destekli gelişmiş skill geliştirme platformu.

![LoL Coach Banner](https://via.placeholder.com/800x200/010A13/C89B3C?text=LoL+Skill+Coach)

## ✨ Özellikler

- 📊 **Detaylı Beceri Analizi** - Mevcut seviyeni değerlendir
- ⚔️ **Mekanik İyileştirme** - Rol bazlı spesifik öneriler
- 🗺️ **Macro Oyun Stratejileri** - Map awareness ve objective control
- 🧠 **Mental Game Tavsiyeleri** - Tilt yönetimi ve consistency
- 📈 **Gelişim Takibi** - LocalStorage ile kayıtlı geçmiş
- 🌍 **Çift Dil Desteği** - Türkçe ve İngilizce
- 🎨 **Profesyonel Tasarım** - LoL temalı modern arayüz
- 💾 **Veri Saklama** - Analizlerini kaydet ve takip et

## 🎯 Nasıl Kullanılır?

1. **Profil Bilgilerini Gir** - Rank, rol, CS/min vb.
2. **Champion Pool Ekle** - Ana oynadığın championları yaz
3. **Sorunlarını Anlat** - Neyi geliştirmek istediğini detaylı yaz
4. **Analiz Et** - AI ile kişiselleştirilmiş tavsiyeleri al
5. **Gelişimini Takip Et** - İstatistiklerin otomatik kaydedilir
